\section{การค้นคว้าข้อมูล}
ในระยะเริ่มต้นของการดำเนินโครงงาน คณะผู้จัดทำได้มุ่งเน้นการสืบค้นและรวบรวมข้อมูลที่เกี่ยวข้อง 
เพื่อสร้างความเข้าใจอย่างถ่องแท้เกี่ยวกับปัญหาที่พบในการจัดการเรียนการสอนวิชาคณิตศาสตร์เรื่องเศษส่วน ตลอดจนศึกษาแนวทางการแก้ไขปัญหาที่มีอยู่ในปัจจุบัน

\subsection{ปัญหาที่มีอยู่ในปัจจุบัน}
คณะผู้จัดทำเกิดข้อสงสัยว่าสิ่งใดเป็นอุปสรรคในการเรียนรู้วิชาคณิตศาสตร์เรื่องเศษส่วนของนักเรียนระดับชั้นประถมศึกษาในปัจจุบัน 
จากการติดต่อสัมภาษณ์นักวิชาการจากสถาบันส่งเสริมการสอนวิทยาศาสตร์และเทคโนโลยี (สสวท.) ได้ข้อมูลว่า 
อุปสรรคในการเรียนรู้วิชาคณิตศาสตร์เรื่องเศษส่วนของนักเรียนระดับประถมศึกษาทั่วโลก เกิดขึ้นมาจากเนื้อหาเรื่องเศษส่วนมีความเป็นนามธรรมสูงและมีรูปแบบที่แตกต่างไปจากจำนวนเต็มที่นักเรียนเคยศึกษามา 
ทำให้หลักการบางอย่างดูขัดแย้งกับความรู้เดิม และเรื่องยากที่จะทำให้ผู้เรียนทุกคนเกิดความเข้าใจที่ถูกต้องพร้อมกันผ่านวิธีการสอนเพียงรูปแบบเดียว

\subsection{วิธีแก้ปัญหาที่มีอยู่ในปัจจุบัน}
จากปัญหาข้างต้น คณะผู้จัดทำได้รวบรวมและศึกษาวิธีการแก้ปัญหาที่ใช้กันอยู่ในปัจจุบัน เพื่อประเมินประสิทธิภาพและข้อจำกัดของแต่ละวิธี โดยมีรายละเอียดดังนี้
\begin{itemize}
    \item การเรียนรู้จากหนังสือเรียน พบว่าหนังสือเรียนส่วนใหญ่มักใช้แบบจำลองพื้นที่ (area model) เช่น รูปภาพเค้ก พิซซ่า หรือตารางสี่เหลี่ยม ในการอธิบายเศษส่วน 
    วิธีนี้สามารถลดปัญหาความเป็นนามธรรมของเศษส่วนได้แต่ส่งผลให้นักเรียนเกิดความยึดติดกับภาพจำเหล่านั้น และประสบปัญหาเมื่อต้องตีความเศษส่วนในรูปแบบอื่น นอกจากนี้ 
    วิธีการดังกล่าวยังเหมาะสมกับการอธิบายเศษส่วนที่มีค่าน้อยกว่า 1 แต่อาจทำให้เกิดความสับสนเมื่อนำไปประยุกต์ใช้กับเนื้อหาที่ซับซ้อนขึ้น 
    เช่น การบวกลบเศษส่วนที่มีค่ามาก เศษเกิน และจำนวนคละ
    \item การใช้สื่อการสอนจากสิ่งรอบตัว เป็นแนวทางที่เน้นให้ผู้เรียนได้ลงมือปฏิบัติจริงและใช้จินตนาการในการแก้ปัญหา 
    เช่น การพับกระดาษแบ่งเป็นส่วน ๆ หรือการเปรียบเทียบความยาวของไม้บรรทัดกับดินสอ 
    วิธีนี้ช่วยลดความยึดติดกับภาพจำและเสริมสร้างความเข้าใจในหลักการของเศษส่วนได้ดี อย่างไรก็ตาม วิธีนี้มีข้อจำกัดด้านความสะดวก 
    เนื่องจากต้องอาศัยเวลาและขั้นตอนในการเตรียมอุปกรณ์ที่ค่อนข้างมาก
    \item การใช้ซอฟต์แวร์ (software) เป็นวิธีการที่สามารถใช้เป็นสื่อการเรียนรู้ได้ทัดเทียมกับวิธีอื่น โดยอาศัยเพียงอุปกรณ์อิเล็กทรอนิกส์ 
    เช่น คอมพิวเตอร์พกพา สมาร์ตโฟน หรือแท็บเล็ต อย่างไรก็ตาม ซอฟต์แวร์เพื่อการเรียนรู้เรื่องเศษส่วนในปัจจุบันยังไม่เป็นที่แพร่หลายนัก 
    และส่วนใหญ่มักรองรับเพียงภาษาต่างประเทศ ทำให้กลายเป็นอุปสรรคในการเข้าถึงสำหรับนักเรียน
\end{itemize}
 
\subsection{ปัญหาที่เหลืออยู่}
จากผลการวิเคราะห์ปัญหาและแนวทางการแก้ไขในปัจจุบัน ทำให้เราเห็นว่า การสร้างความเข้าใจและทำให้ผู้เรียนมองเห็นภาพที่ถูกต้องของเศษส่วนยังคงเป็นความท้าทายหลัก นอกจากนี้ วิธีการแก้ปัญหาที่มีอยู่บางส่วนยังมีข้อจำกัดด้านการเข้าถึงและความสะดวกในการใช้งาน ทำให้ยังไม่สามารถแก้ไขปัญหาได้อย่างมีประสิทธิภาพเท่าที่ควร

\section{Survey 1: problem survey}
จากปัญหาที่เกิดขึ้นจริงในการเรียนการสอนวิชาคณิตศาสตร์เรื่องเศษส่วน คณะผู้จัดทำจึงได้ทำการลงพื้นครั้งที่ 1 มีจุดประสงค์เพื่อยืนยันปัญหา
และเก็บรวบรวมข้อมูลเชิงลึกเกี่ยวกับการเรียนรู้และกระบวนการเรียนการสอนวิชาคณิตศาสตร์เรื่องเศษส่วนที่เกิดขึ้นจริง โดยในแต่ละพื้นที่จะดำเนินการสัมภาษณ์กับบุคคลคน 2 กลุ่ม ได้แก่
\begin{itemize}
    \item นักเรียน จำนวน 6 คน: มุ่งเน้นการสอบถามเกี่ยวกับกระบวนการคิดเพื่อประเมินระดับความเข้าใจเรื่องเศษส่วน สอบถามความรู้สึกและทัศนคติที่มีต่อวิชาคณิตศาสตร์ รวมถึงสำรวจความชื่นชอบในการใช้งานอุปกรณ์อิเล็กทรอนิกส์ เพื่อนำข้อมูลมาใช้ประกอบการตัดสินใจเลือกแพลตฟอร์มหลักสำหรับการพัฒนาโครงงาน
    \item ครูผู้สอน จำนวน 1 คน: มุ่งเน้นการสอบถามเกี่ยวกับวิธีการและเทคนิคการจัดการเรียนการสอน เพื่อรวบรวมข้อมูลเกี่ยวกับรูปแบบการสอนที่มีประสิทธิภาพจากประสบการณ์ตรงของผู้สอน รวมถึงสำรวจประเภทของสื่อและอุปกรณ์ที่ใช้ในชั้นเรียน เพื่อนำมาเป็นข้อมูลอ้างอิงในการออกแบบระบบ
\end{itemize}

ในส่วนของพื้นที่ คณะผู้จัดทำได้คัดเลือกโรงเรียนเป้าหมายที่เปิดสอนระดับชั้นประถมศึกษาตอนปลายในเขตพื้นที่จังหวัดเชียงใหม่ จำนวน 6 แห่ง 
ได้แก่ โรงเรียนพิงครัตน์ โรงเรียนพุทธิโสภน โรงเรียนดาราวิทยาลัย โรงเรียนบ้านเชิงดอยสุเทพ โรงเรียนโกวิทธำรงเชียงใหม่ และโรงเรียนสาธิตมหาวิทยาลัยเชียงใหม่ 
โดยเกณฑ์ในการคัดเลือกพิจารณาจากระยะทางที่สะดวกต่อการเดินทางจากมหาวิทยาลัยเชียงใหม่ และความเป็นไปได้ในการติดต่อขออนุญาตลงพื้นที่เก็บข้อมูลจากทางสถานศึกษา

\section{แนวคิดการออกแบบเนื้อหาการเรียนรู้}
เนื้อหาการเรียนรู้ใน web application นี้ถูกออกแบบโดยอิงจากหลักสูตรคณิตศาสตร์ของ สสวท. สำหรับระดับชั้นประถมศึกษา และการสัมภาษณ์ครูผู้สอนที่ได้ลงพื้นที่สำรวจในวิชา 269491
โดยมีการผสมผสานและจัดลำดับเนื้อหาให้มีการเชื่อมโยงกันระหว่างการวัดและเศษส่วน เพื่อนำเสนอเนื้อหาเศษส่วนออกมาในรูปแบบ Length model

\section{User Flow}
User flow ของ web application นี้ถูกออกแบบมาเพื่อให้ผู้ใช้งานสามารถเรียนรู้เรื่องเศษส่วนได้อย่างมีประสิทธิภาพและสนุกสนาน โดยมีการจัดลำดับเนื้อหาดังนี้
\begin{itemize}
    \item การสร้างบัญชีผู้ใช้ใหม่ หรือการเข้าสู่ระบบด้วยบัญชีที่มีอยู่แล้ว เพื่อให้ผู้ใช้งานสามารถบันทึกความคืบหน้าในการเรียนรู้ได้
    \item การเลือกบทเรียนที่ต้องการเรียนรู้ โดยจำนวนบทเรียนที่เข้าถึงได้จะเพิ่มขึ้นตามความคืบหน้าของผู้ใช้งาน
    \item การใช้งานซ้ำเพื่อทบทวนเนื้อหาที่เคยเรียนมาแล้ว หรือการเลือกบทเรียนที่ยังไม่เคยเรียนเพื่อเพิ่มความรู้ใหม่
\end{itemize}

\section{ฟีเจอร์}
ในส่วนนี้จะอธิบายเกี่ยวกับหลักการทำงานของฟีเจอร์ต่างๆที่มีอยู่ใน web application
\subsection{การสร้างบัญชี}
การสร้างบัญชีเป็นฟีเจอร์ที่ทำให้ผู้ใช้สามารถสร้างบัญชีสำหรับใช้งาน web application โดยบัญชีนี้จะบันทึกข้อมูลความคืบหน้าการเรียนรู้ของผู้ใช้ 
ทำให้ผู้ใช้สามารถหยุดพักได้ทุกเมื่อและสามารถกลับเข้ามาเรียนรู้ต่อจากครั้งก่อนได้ 
นอกจากนี้ ผู้ใช้สามารถสร้างบัญชีได้หลายบัญชี เพื่อรองรับกรณีที่ผู้ใช้ต้องการให้บุคคลอื่นใช้งานระบบ

\subsection{การเลือกบทเรียน}
การเลือกบทเรียนเป็นฟีเจอร์ที่ทำให้ผู้ใช้งานสามารถเลือกบทเรียนที่ผู้ใช้ต้องการได้ผ่านหน้า web   
โดยบทเรียนที่ผู้ใช้สามารถเลือกได้คือบทเรียนที่ผู้ใช้งานเคยเรียนผ่านมาแล้ว หรือบทเรียนต่อไปที่ผู้ใช้งานจะต้องเรียน ซึ่งจะแสดงเป็นปุ่มสีเข้ม 
ส่วนบทเรียนที่ผู้เล่นยังไม่สามารถเข้าเรียนได้จะแสดงเป็นสีอ่อน

 \section{เนื้อหาในบทเรียน}
เนื้อหาในบทเรียนถูกออกแบบมาเพื่อให้ผู้ใช้งานสามารถเรียนรู้เรื่องเศษส่วน ซึ่งเนื้อหาจะมีความเชื่อมโยงกันและค่อยๆเพิ่มความซับซ้อนขึ้นตามลำดับ
โดยเนื้อหาทั้งหมดในบทเรียนจะถูกแบ่งออกเป็น 9 episode ซึ่งแต่ละ episode จะมีเนื้อหาดังนี้
\begin{itemize}
    \item Episode 1: การวัดความยาวและการเปรียบเทียบความยาว
    ผู้ใช้งานจะได้เรียนรู้เกี่ยวกับการวัดและการเปรียบเทียบความยาวของดินสอแต่ละแท่ง โดยบทเรียนนี้มีเป้าหมายให้ผู้ใช้งานได้เข้าใจและรู้จักกับการวัดที่ถูกต้อง
    \item Episode 2: การวัดความยาวด้วยหน่วยที่ไม่ใช่หน่วยมาตรฐาน
    ผู้ใช้งานจะได้ทำการวัดความยาวของหนังสือเล่มหนึ่งด้วยดินสอแต่ละแท่งที่มีความยาวไม่เท่ากัน ซึ่งถือเป็นการวัดโดยใช้หน่วยที่ไม่ใช่หน่วยมาตรฐาน โดยบทเรียนนี้มีเป้าหมายให้ผู้ใช้งานได้เข้าใจว่าการวัดความยาวควรมีการใช้หน่วยการวัดที่เป็นมาตรฐานเพื่อให้ได้ผลลัพธ์ที่ถูกต้องและสามารถเปรียบเทียบกันได้
    \item Episode 3: การวัดความยาววัตถุด้วยหน่วยมาตรฐาน
    ผู้ใช้งานจะได้ทำการวัดความยาวของวัตถุต่างๆที่มีความยาวเป็นจำนวนเต็มด้วยดินสอที่มีความยาวเป็นมาตรฐาน โดยบทเรียนนี้มีเป้าหมายให้ผู้ใช้งานได้คุ้นเคยกับการใช้หน่วยวัดที่เป็นมาตรฐาน
    \item Episode 4: การวัดความยาววัตถุที่มีความยาวเป็นเศษส่วน
    ผู้ใช้งานจะได้ทำการวัดความยาวของวัตถุต่างๆที่มีความยาวเป็นเศษส่วน เพื่อให้ผู้ใช้งานได้เห็นภาพของความยาวที่เป็นเศษส่วน และเข้าใจถึงความสำคัญและที่มาของเศษส่วน
    \item Episode 5: นิยามของเศษส่วน
    ผู้ใช้งานจะได้เรียนรู้เกี่ยวกับนิยามของเศษส่วน ด้วยการใช้คำว่าครึ่งและการวัดความยาวมาแสดงภาพให้เห็นว่าเศษส่วนมีคุณสมบัติอย่างไร โดยบทเรียนนี้มีเป้าหมายให้ผู้ใช้งานได้เข้าใจว่าเศษส่วนคืออะไร
    \item Episode 6: เศษส่วนในเส้นจำนวน
    ผู้ใช้งานจะได้เรียนรู้เกี่ยวกับการแสดงเศษส่วนบนเส้นจำนวน ด้วยการนำความรู้เกี่ยวกับการวัดในบทเรียนก่อนหน้ามาวางบนเส้นจำนวน เพื่อให้ผู้ใช้งานได้เห็นภาพของเศษส่วนในรูปแบบที่แตกต่างออกไป และเข้าใจว่าเศษส่วนสามารถแสดงออกมาในรูปแบบต่างๆได้
    \item Episode 7: การบวกเศษส่วนที่มีตัวส่วนเท่ากัน
    ผู้ใช้งานจะได้เรียนรู้เกี่ยวกับการบวกเศษส่วนที่มีตัวส่วนเท่ากัน ซึ่งจะมีการแสดงเศษส่วนบนเส้นจำนวนเพื่อช่วยให้ผู้ใช้งานเห็นภาพของการบวกเศษส่วน และเข้าใจว่าการบวกเศษส่วนที่มีตัวส่วนเท่ากันนั้นสามารถทำได้โดยการบวกตัวเศษเข้าด้วยกันได้เลย
    \item Episode 8: การแปลงค่าเศษส่วน
    ผู้ใช้งานจะได้เรียนรู้เกี่ยวกับการแปลงค่าเศษส่วน ซึ่งเนื้อหาในบทเรียนนี้จะมีการใช้เส้นจำนวนประกอบเพื่อช่วยให้ผู้ใช้งานเห็นภาพของการแปลงค่าเศษส่วน และเข้าใจว่าการแปลงค่าเศษส่วนนั้นสามารถทำได้โดยการคูณหรือหารตัวเศษและตัวส่วนด้วยจำนวนเดียวกัน
    \item Episode 9: การบวกเศษส่วนที่มีตัวส่วนไม่เท่ากัน
    ผู้ใช้งานจะได้เรียนรู้เกี่ยวกับการบวกเศษส่วนที่มีตัวส่วนไม่เท่ากัน ซึ่งเนื้อหาในบทเรียนนี้จะมีใช้เนื้อหาในบทก่อนๆเพื่อช่วยให้ผู้ใช้งานเข้าใจว่าการบวกเศษส่วนที่มีตัวส่วนไม่เท่ากันนั้นสามารถทำได้โดยการแปลงค่าเศษส่วนให้มีตัวส่วนเท่ากันก่อนแล้วจึงค่อยบวกตัวเศษเข้าด้วยกัน
\end{itemize}

\section{ขั้นตอนการทำงานของระบบ}
web application นี้เริ่มต้นทำงานเมื่อผู้ใช้เปิดใช้งาน web application ผ่าน web browser จากนั้นระบบจะให้ผู้ใช้ทำการ login เพื่อเข้าสู่การเรียนรู้เนื้อหาที่คณะผู้จัดทำได้เตรียนไว้ 
โดยการ login สามารถทำได้ 2 แบบดังนี้
\begin{enumerate}
    \item ไม่มีบัญชีผู้ใช้: ผู้ใช้เข้าใช้งาน web application นี้เป็นครั้งแรก ระบบจะให้ผู้ใช้สร้างบัญชีผู้ใช้ใหม่และเข้าสู่บนนำเพื่อเรียนรู้เกี่ยวกับพื้นฐานการใช้งาน web application นี้
    \item มีบัญชีผู้ใช้: ระบบจะแสดงบัญชีที่เคยถูกสร้างไว้ ผู้ใช้สามารถ login ด้วยบัญชีนั้นและเข้าสู่การเรียนรู้ได้ทันที
\end{enumerate}
เมื่อผู้ใช้ทำการ login เรียบร้อยแล้วผู้ใช้จะเข้าสู่หน้า map ที่แสดงความคืบหน้าในการเรียนรู้ของผู้ใช้และแสดงบทเรียนทั้งหมดที่มีอยู่ในระบบ โดยบทเรียนจะถูกแบ่งเป็น episode 
ผู้ใช้สามารถเลือกเนื้อหาที่ต้องการเรียนรู้ได้ด้วยตนเอง หลังจากเลือกเนื้อหาเรียบร้อยแล้วผู้ใช้จะเข้าสู่การเรียนรู้บทเรียนที่ถูกเลือก ระบบจะทำการแสดงเนื้อหาให้ผู้ใช้อ่านหรือตอบโต้เพื่อเรียนรู้เนื้อหาบนหน้า web 
โดยระบบจะตรวจสอบการตอบโต้ของผู้ใช้ ซึ่งการตอบโต้ที่ถูกต้องของผู้ใช้จะทำให้เนื้อหาการเรียนรู้ดำเนินต่อไปได้ การตอบโต้ที่ไม่ถูกต้องจะทำให้เนื้อหาไม่ดำเนินต่อแต่ระบบจะให้โอกาสผู้ใช้ได้แก้การตอบโต้นั้นจนกว่าผู้ใช้จะทำการตอบโต้ที่ถูกต้อง 
เมื่อผู้ใช้เรียนรู้เนื้อหาของบทเรียนหนึ่งจบแล้ว ระบบจะบันทึกความคืบหน้าในการเรียนรู้ของผู้ใช้และแสดงตัวเลือกหลังจบบทเรียนที่ผู้ใช้สามารถเลือกได้ว่า ผู้ใช้ต้องการที่จะไปที่หน้า map เพื่อหยุดพัก
