\section{การประเมินระบบโดยการลงพื้นที่}
เนื่องจากโครงงานนี้เป็นการพัฒนาสื่อการเรียนรู้สำหรับเด็ก คณะผู้จัดทำจึงเน้นไปที่การลงพื้นที่ โรงเรียนที่มีการสอนระดับประถมศึกษาตอนปลาย ในจังหวัดเชียงใหม่เพื่อทำการประเมินระบบ
โดยทําการเก็บข้อมูล ความคิดเห็น ข้อเสนอแนะจากนักเรียนในทุกๆครั้งที่ลงพื้นที่เพื่อนําไปพัฒนาแอปพลิเคชั่นสําหรับการลงพื้นที่ครั้งถัดไป 
ภายในระยะเวลาการทําโครงงานกลุ่มของเราได้ทําการลงพื้นที่ทั้งสิ้น 6 ครั้ง โดยแบ่งเป็น 3 ช่วง และกลุ่มตัวอย่างที่ใช้ในการทดสอบระบบและประเมินผล คือ นักเรียนระดับชั้นประถมศึกษาปีที่ 4 จำนวน 33 คน

\section{รูปแบบและขั้นตอนการประเมิน}
การประเมินผลถูกแบ่งออกเป็น 2 ส่วนหลัก ได้แก่ การทดสอบการใช้งานระบบ (usability test) และการประเมินผลสัมฤทธิ์ทางการเรียน (pre-test และ post-test) โดยมีขั้นตอนการดำเนินการดังนี้

\subsection{การทดสอบการใช้งานระบบ (usability test)}
ให้นักเรียนกลุ่มตัวอย่างทดลองใช้งานและโต้ตอบกับ web application เพื่อทดสอบฟังก์ชันการทำงานพื้นฐานของระบบ (functional testing) และประเมินความสะดวกในการใช้งาน (usability) โดยในระหว่างการทดสอบ 
คณะผู้จัดทำจะทำการสังเกตพฤติกรรม การตอบสนอง และบันทึกปัญหาที่พบระหว่างการใช้งาน เพื่อนำข้อมูลไปใช้ในการปรับปรุงส่วน UX/UI ให้เหมาะสมกับผู้เรียนมากยิ่งขึ้น

\subsection{แนวทางการประเมินผลสัมฤทธิ์ทางการเรียน (pre-test และ post-test)}
เพื่อวัดประสิทธิภาพของสื่อการสอนว่าสามารถช่วยเสริมสร้างความเข้าใจในเรื่องเศษส่วนได้จริงหรือไม่ คณะผู้จัดทำได้กำหนดขั้นตอนการทดสอบความรู้ ดังนี้
\begin{enumerate}
    \item \textbf{การทดสอบก่อนเรียน (pre-test):} ให้นักเรียนทำโจทย์ปัญหาเพื่อวัดระดับความเข้าใจพื้นฐานเกี่ยวกับเศษส่วนก่อนการใช้งานสื่อการสอน
    \item \textbf{การทดลองใช้งานระบบ:} ให้นักเรียนเรียนรู้และทดลองใช้งาน web application เพื่อทำความเข้าใจเศษส่วนผ่านระบบ sandbox
    \item \textbf{การตรวจสอบความเข้าใจเชิงลึก:} ประเมินความสามารถของนักเรียนผ่านการตั้งคำถามและการให้อธิบายเหตุผล ในประเด็นต่อไปนี้
        \begin{itemize}
            \item การวาดรูปแสดงเศษส่วนในรูปแบบ length model
            \item การอธิบายหลักการเปรียบเทียบเศษส่วนพร้อมระบุเหตุผล
            \item การอธิบายการบวกเศษส่วนที่ตัวส่วนไม่เท่ากัน พร้อมให้เหตุผลถึงความจำเป็นในการทำตัวส่วนให้เท่ากันก่อน
            \item การระบุตำแหน่งของเศษส่วน (เช่น $\frac{1}{4}$) บนเส้นจำนวนได้อย่างถูกต้อง
        \end{itemize}
    \item \textbf{การทดสอบหลังเรียน (post-test):} ให้นักเรียนทำโจทย์ปัญหาชุดเดิมหรือเทียบเท่า เพื่อประเมินผลการเรียนรู้และวัดความก้าวหน้าหลังจากผ่านการใช้งานสื่อการสอน
\end{enumerate}

\section{สรุปแนวทางการประเมิน}
การประเมิน web application ด้วยรูปแบบดังกล่าว ช่วยให้คณะผู้จัดทำสามารถตรวจสอบและยืนยันได้ว่าสื่อการสอนที่พัฒนาขึ้นสามารถสนับสนุนและส่งเสริมการเรียนรู้เรื่องเศษส่วนได้จริง 
ผ่านการเปรียบเทียบระดับความเข้าใจของนักเรียนก่อนและหลังการใช้งาน ควบคู่ไปกับการประเมินประสิทธิภาพการทำงานของระบบจากการสังเกตพฤติกรรมการใช้งานจริงได้อย่างครบถ้วนและชัดเจน

\section{ผลลัพท์จากการประเมินระบบ}
จากการประเมินระบบผ่านการลงพื้นที่ทดสอบกับนักเรียนจริง คณะผู้จัดทำได้ทำการรวบรวมข้อมูลโดยมีผลลัพธ์จากนักเรียนทั้งหมด 33 คนดังนี้
\begin{itemize}
    \item ความเข้าใจของนักเรียนที่มีต่อคำบรรยาย: นักเรียนจำนวน 33 คนสามารถเข้าใจคำบรรยายที่ระบบให้มาได้อย่างชัดเจน โดยนักเรียนสามารถใช้งานระบบและตอบคำถามที่เกี่ยวข้องกับเนื้อหาได้อย่างถูกต้องจากการอ่านคำบรรยายที่ระบบให้มา
    \item ฟังก์ชั่นการลากวางและการเลือกคำตอบ: นักเรียนจำนวน 33 คนสามารถใช้งานฟังก์ชั่นการลากวางและการเลือกคำตอบได้ โดยไม่มีปัญหาในการโต้ตอบกับระบบ และสามารถทำกิจกรรมต่าง ๆ ที่ระบบกำหนดได้ครบถ้วนตามกรอบที่ตั้งไว้
    \item ความรู้สึกของนักเรียนที่มีต่อ UI: นักเรียนจำนวน 29 คนให้การตอบรับที่ดีโดยนักเรียนรู้สึกว่า UI ของระบบมีความชัดเจนและสวยงาม ในขณะที่นักเรียนจำนวน 4 คนให้การตอบรับในเชิงลบ โดยนักเรียนรู้สึกว่า UI ของระบบไม่น่าใช้งานและไม่ดึงดูดความสนใจ
\end{itemize}

\begin{itemize}
    \item จากผลการทดสอบก่อนเรียน (pre-test) ได้ผลดังนี้
        \subitem การวาดรูปแสดงเศษส่วนในรูปแบบ length model: เนื่องด้วยเวลาที่จำกัดจึงทำให้การทดสอบในหัวข้อดังกล่าวมีนักเรียนทำการทดสอบรวม 6 คนเท่านั้น โดยนักเรียนจำนวน 2 คนสามารถวาดรูปแสดงเศษส่วนในรูปแบบ length model ได้อย่างถูกต้อง ในขณะที่นักเรียนจำนวน 4 คนสามารถวาดรูปแสดงเศษส่วนในรูปแบบอื่นๆที่ไม่ใช่ length model ได้
        \subitem การอธิบายหลักการเปรียบเทียบเศษส่วนพร้อมระบุเหตุผล: นักเรียนจำนวน 12 คนสามารถเปรียบเทียบเศษส่วนได้อย่างถูกต้อง แต่ไม่มีนักเรียนที่สามารถระบุเหตุผลที่ถูกต้องได้ โดยนักเรียนไม่สามารถให้เหตุผลได้เลย
        \subitem การอธิบายการบวกเศษส่วนที่ตัวส่วนไม่เท่ากัน พร้อมให้เหตุผลถึงความจำเป็นในการทำตัวส่วนให้เท่ากันก่อน: นักเรียนจำนวน 12 คนสามารถบวกเศษส่วนที่ตัวส่วนไม่เท่ากันได้อย่างถูกต้อง แต่ไม่มีนักเรียนที่สามารถระบุเหตุผลที่ถูกต้องได้ โดยนักเรียนให้เหตุผลเพียงว่า "คุณครูสอนมาแบบนี้" หรือ "สูตรให้ทำแบบนี้"
        \subitem การระบุตำแหน่งของเศษส่วน (เช่น $\frac{1}{4}$) บนเส้นจำนวนได้อย่างถูกต้อง: นักเรียนจำนวน 12 คนระบุตำแหน่งของเศษส่วนบนเส้นจำนวนได้ไม่ถูกต้อง โดยนักเรียนบางคนได้ให้เหตุผลเพิ่มเติมว่า "ไม่เคยเรียนแบบนี้มาก่อน"
    \item จากผลการทดสอบหลังเรียน (post-test) ได้ผลดังนี้
        \subitem การวาดรูปแสดงเศษส่วนในรูปแบบ length model: นักเรียนทั้ง 6 คนสามารถวาดรูปแสดงเศษส่วนในรูปแบบ length model ได้อย่างถูกต้อง
        \subitem การอธิบายหลักการเปรียบเทียบเศษส่วนพร้อมระบุเหตุผล: นักเรียนจำนวน 12 คนสามารถเปรียบเทียบเศษส่วนได้อย่างถูกต้อง โดยนักเรียนจำนวน 12 คนสามารถให้เหตุผลที่ถูกต้องตามความเข้าใจของตนเองได้
        \subitem การอธิบายการบวกเศษส่วนที่ตัวส่วนไม่เท่ากัน พร้อมให้เหตุผลถึงความจำเป็นในการทำตัวส่วนให้เท่ากันก่อน: นักเรียนจำนวน 12 คนสามารถบวกเศษส่วนที่ตัวส่วนไม่เท่ากันได้อย่างถูกต้อง โดยนักเรียนจำนวน 12 คนสามารถให้เหตุผลที่ถูกต้องตามความเข้าใจของตนเองได้
        \subitem การระบุตำแหน่งของเศษส่วน (เช่น $\frac{1}{4}$) บนเส้นจำนวนได้อย่างถูกต้อง: นักเรียนจำนวน 6 คนสามารถระบุตำแหน่งของเศษส่วนบนเส้นจำนวนได้อย่างถูกต้อง นักเรียนจำนวน 2 คนสามารถบอกได้ว่าคำตอบของตนเองในการทดสอบก่อนเรียน (pre-test) นั้นไม่ถูกต้อง และนักเรียนจำนวน 4 คนระบุตำแหน่งของเศษส่วนบนเส้นจำนวนได้ไม่ถูกต้อง
\end{itemize}

\section{สรุปผลการประเมิน}
จากผลลัพธ์ที่ได้จากการประเมินระบบและผลสัมฤทธิ์ทางการเรียน คณะผู้จัดทำสามารถสรุปได้ว่า web application ที่พัฒนาขึ้นสามารถสนับสนุนการเรียนรู้เรื่องเศษส่วนได้อย่างมีประสิทธิภาพ โดยนักเรียนสามารถเข้าใจคำบรรยายและใช้งานฟังก์ชั่นต่าง ๆ ของระบบได้อย่างถูกต้อง
และได้รับการตอบรับที่ดีในเรื่องของความน่าสนใจในการใช้งาน นอกจากนี้ ผลการประเมินผลสัมฤทธิ์ทางการเรียนยังแสดงให้เห็นว่าหลังจากผ่านการใช้งานสื่อการสอน นักเรียนมีความเข้าใจในเรื่องเศษส่วนที่ดีขึ้นอย่างมีนัยสำคัญ โดยสามารถวาดรูปแสดงเศษส่วนในรูปแบบ length model ได้อย่างถูกต้อง
และสามารถอธิบายหลักการเปรียบเทียบและการบวกเศษส่วนได้อย่างถูกต้องพร้อมให้เหตุผลที่ถูกต้องตามความเข้าใจของตนเองได้ นอกจากนี้ นักเรียนบางคนยังสามารถระบุตำแหน่งของเศษส่วนบนเส้นจำนวนได้อย่างถูกต้องอีกด้วย ซึ่งเป็นผลลัพธ์ที่สะท้อนให้เห็นถึงความเข้าใจที่เพิ่มขึ้นในเรื่องเศษส่วนของนักเรียน
