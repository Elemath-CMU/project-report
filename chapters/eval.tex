\section{การประเมินระบบโดยการลงพื้นที่}
เนื่องจากโครงงานนี้เป็นการพัฒนาสื่อการเรียนรู้สำหรับเด็ก คณะผู้จัดทำจึงเน้นไปที่การลงพื้นที่ โรงเรียนที่มีการสอนระดับประถมศึกษาตอนปลาย ในจังหวัดเชียงใหม่เพื่อทำการประเมินระบบ
โดยทําการเก็บข้อมูล ความคิดเห็น ข้อเสนอแนะจากนักเรียนในทุกๆครั้งที่ลงพื้นที่เพื่อนําไปพัฒนาแอปพลิเคชั่นสําหรับการลงพื้นที่ครั้งถัดไป 
ภายในระยะเวลาการทําโครงงานกลุ่มของเราได้ทําการลงพื้นที่ทั้งสิ้น 6 ครั้ง และกลุ่มตัวอย่างที่ใช้ในการทดสอบระบบและประเมินผล คือ นักเรียนระดับชั้นประถมศึกษาปีที่ 4 จำนวน 33 คน
โดยมีรายละเอียดการประเมินระบบดังนี้
\begin{itemize}
    \item ลงพื้นที่ครั้งที่ 1 (23 กุมภาพันธ์ 2569) : ทดสอบระบบการใช้งานที่โรงเรียนสาธิตมหาวิทยาลัยเชียงใหม่ โดยมีนักเรียนจำนวน 6 คนเข้าร่วมการทดสอบระบบ
    โดยการลงพื้นที่ในครั้งนี้ได้มีการทดสอบการใช้งานระบบ (Usability Test) และการสอบถามความคิดเห็นและข้อเสนอแนะจากนักเรียนเพื่อปรับปรุงระบบในครั้งถัดไป
    \item ลงพื้นที่ครั้งที่ 2 (24 กุมภาพันธ์ 2569) : ทดสอบระบบการใช้งานที่โรงเรียนดาราวิทยาลัย โดยมีนักเรียนจำนวน 6 คนเข้าร่วมการทดสอบระบบ
    โดยการลงพื้นที่ในครั้งนี้ได้มีการทดสอบการใช้งานระบบ (Usability Test) และการสอบถามความคิดเห็นและข้อเสนอแนะจากนักเรียนเพื่อปรับปรุงระบบในครั้งถัดไป
    \item ลงพื้นที่ครั้งที่ 3 (26 กุมภาพันธ์ 2569) : ทดสอบระบบการใช้งานที่โรงเรียนโกวิทธำรงเชียงใหม่ โดยมีนักเรียนจำนวน 6 คนเข้าร่วมการทดสอบระบบ
    โดยการลงพื้นที่ในครั้งนี้ได้มีการปรับปรุง UI/UX ของแอปพลิเคชั่นจากการลงพื้นที่ 2 ครั้งที่ผ่านมา และได้ทำการทดสอบการใช้งานระบบ (Usability Test) และการสอบถามความคิดเห็นและข้อเสนอแนะจากนักเรียนเพื่อปรับปรุงระบบในครั้งถัดไป
    \item ลงพื้นที่ครั้งที่ 4 (27 กุมภาพันธ์ 2569) : ทดสอบระบบการใช้งานที่โรงเรียนพิงครัตน์ โดยมีนักเรียนจำนวน 3 คนเข้าร่วมการทดสอบระบบ
    โดยการลงพื้นที่ในครั้งนี้ได้มีการทดสอบการใช้งานระบบ (Usability Test) และการสอบถามความคิดเห็นและข้อเสนอแนะจากนักเรียนเพื่อปรับปรุงระบบในครั้งถัดไป
    \item ลงพื้นที่ครั้งที่ 5 (6 มีนาคม 2569) : ทดสอบระบบการใช้งานที่โรงเรียนบ้านเชิงดอยสุเทพ โดยมีนักเรียนจำนวน 6 คนเข้าร่วมการทดสอบระบบ
    โดยการลงพื้นที่ในครั้งนี้ แอปพลิเคชั่นได้รับการอัพเดตเนื้อหาครบถ้วนแล้ว จึงทำให้ได้มีการทดสอบการใช้งานระบบ (Usability Test) และการประเมินผลสัมฤทธิ์ทางการเรียน (Pre-test และ Post-test) เพื่อประเมินผลการเรียนรู้ของนักเรียนหลังจากผ่านการใช้งานสื่อการสอน
    \item ลงพื้นที่ครั้งที่ 6 (17 มีนาคม 2569) : ทดสอบระบบการใช้งานที่โรงเรียนพุทธิโศภน โดยมีนักเรียนจำนวน 6 คนเข้าร่วมการทดสอบระบบ
    โดยการลงพื้นที่ในครั้งนี้ได้มีการทดสอบการใช้งานระบบ (Usability Test) และการประเมินผลสัมฤทธิ์ทางการเรียน (Pre-test และ Post-test) เพื่อประเมินผลการเรียนรู้ของนักเรียนหลังจากผ่านการใช้งานสื่อการสอน
\end{itemize}


\section{รูปแบบและขั้นตอนการประเมิน}
การประเมินผลถูกแบ่งออกเป็น 2 ส่วนหลัก ได้แก่ การทดสอบการใช้งานระบบ (Usability Test) และการประเมินผลสัมฤทธิ์ทางการเรียน (Pre-test และ Post-test) โดยมีขั้นตอนการดำเนินการดังนี้

\subsection{การทดสอบการใช้งานระบบ (Usability Test)}
ให้นักเรียนกลุ่มตัวอย่างทดลองใช้งานและโต้ตอบกับ Web Application เพื่อทดสอบฟังก์ชันการทำงานพื้นฐานของระบบ (Functional Testing) และประเมินความสะดวกในการใช้งาน (Usability) โดยในระหว่างการทดสอบ 
คณะผู้จัดทำจะทำการสังเกตพฤติกรรม การตอบสนอง และบันทึกปัญหาที่พบระหว่างการใช้งาน เพื่อนำข้อมูลไปใช้ในการปรับปรุงส่วน UX/UI ให้เหมาะสมกับผู้เรียนมากยิ่งขึ้น

\subsection{การประเมินผลสัมฤทธิ์ทางการเรียน (Pre-test และ Post-test)}
เพื่อวัดประสิทธิภาพของสื่อการสอนว่าสามารถช่วยเสริมสร้างความเข้าใจในเรื่องเศษส่วนได้จริงหรือไม่ คณะผู้จัดทำได้กำหนดขั้นตอนการทดสอบความรู้ ดังนี้
\begin{enumerate}
    \item \textbf{การทดสอบก่อนเรียน (Pre-test):} ให้นักเรียนทำโจทย์ปัญหาเพื่อวัดระดับความเข้าใจพื้นฐานเกี่ยวกับเศษส่วนก่อนการใช้งานสื่อการสอน
    \item \textbf{การทดลองใช้งานระบบ:} ให้นักเรียนเรียนรู้และทดลองใช้งาน Web Application เพื่อทำความเข้าใจเศษส่วนผ่านระบบ Sandbox
    \item \textbf{การตรวจสอบความเข้าใจเชิงลึก:} ประเมินความสามารถของนักเรียนผ่านการตั้งคำถามและการให้อธิบายเหตุผล ในประเด็นต่อไปนี้
        \itemize
            \item การวาดรูปแสดงเศษส่วนในรูปแบบ Length Model
            \item การอธิบายหลักการเปรียบเทียบเศษส่วนพร้อมระบุเหตุผล
            \item การอธิบายการบวกเศษส่วนที่ตัวส่วนไม่เท่ากัน พร้อมให้เหตุผลถึงความจำเป็นในการทำตัวส่วนให้เท่ากันก่อน
            \item การระบุตำแหน่งของเศษส่วน (เช่น $\frac{1}{4}$) บนเส้นจำนวนได้อย่างถูกต้อง
        \end{itemize}
    \item \textbf{การทดสอบหลังเรียน (Post-test):} ให้นักเรียนทำโจทย์ปัญหาชุดเดิมหรือเทียบเท่า เพื่อประเมินผลการเรียนรู้และวัดความก้าวหน้าหลังจากผ่านการใช้งานสื่อการสอน
\end{enumerate}

\section{สรุปแนวทางการประเมิน}
การประเมิน Web Application ด้วยรูปแบบดังกล่าว ช่วยให้คณะผู้จัดทำสามารถตรวจสอบและยืนยันได้ว่าสื่อการสอนที่พัฒนาขึ้นสามารถสนับสนุนและส่งเสริมการเรียนรู้เรื่องเศษส่วนได้จริง 
ผ่านการเปรียบเทียบระดับความเข้าใจของนักเรียนก่อนและหลังการใช้งาน ควบคู่ไปกับการประเมินประสิทธิภาพการทำงานของระบบจากการสังเกตพฤติกรรมการใช้งานจริงได้อย่างครบถ้วนและชัดเจน
