\section{\ifenglish Project rationale\else ที่มาของโครงงาน\fi}
ในปัจจุบัน การเรียนรู้คณิตศาสตร์เป็นสิ่งสำคัญที่ช่วยพัฒนาทักษะการคิดวิเคราะห์ของผู้เรียน เนื่องจากคณิตศาสตร์เป็นพื้นฐานของการใช้เหตุผล การแก้ปัญหา 
และการเชื่อมโยงความรู้ไปสู่ศาสตร์อื่น ๆ อย่างไรก็ตาม จากผลการประเมินสมรรถนะนักเรียนระดับนานาชาติ (PISA 2022) พบว่า 
คะแนนเฉลี่ยของนักเรียนไทยต่ำกว่าค่าเฉลี่ยของประเทศสมาชิก OECD ทุกวิชา โดยเฉพาะในวิชาคณิตศาสตร์และการอ่าน ซึ่งมีคะแนนไม่ถึง 400 คะแนน 
สะท้อนให้เห็นถึงปัญหาการทำความเข้าใจแนวคิดพื้นฐานและการคิดเชิงวิเคราะห์ของผู้เรียนไทยในปัจจุบัน

เมื่อศึกษาข้อมูลเพิ่มเติมและพูดคุยกับนักวิชาการด้านการศึกษาจากสถาบันส่งเสริมการสอนวิทยาศาสตร์และเทคโนโลยี (สสวท.) พบว่า หนึ่งในหัวข้อที่นักเรียนทั่วโลก 
รวมทั้งประเทศไทยมักประสบปัญหาคือ เศษส่วน แม้จะเป็นหัวข้อพื้นฐาน แต่แนวคิดของเศษส่วนเป็นนามธรรมและเข้าใจได้ยาก เด็กจำนวนมากสามารถคำนวณได้ 
แต่ไม่สามารถอธิบายได้ว่าทำไมจึงคิดเช่นนั้น ซึ่งส่งผลให้ไม่เข้าใจแก่นของคณิตศาสตร์อย่างแท้จริง

เพื่อยืนยันปัญหานี้ พวกเราได้ลงพื้นที่เก็บข้อมูลจากโรงเรียนในเขตพื้นที่ใกล้มหาวิทยาลัยเชียงใหม่ จำนวน 6 โรงเรียน โดยเลือกโรงเรียนที่มีการสอนระดับประถมศึกษาตอนปลาย 
ซึ่งเป็นช่วงวัยที่เริ่มเรียนรู้เรื่องเศษส่วน การเก็บข้อมูลประกอบด้วยการสัมภาษณ์นักเรียนชั้นประถมศึกษาปีที่ 5 จำนวน 6 คน และครูผู้สอนวิชาคณิตศาสตร์ 1 คนต่อโรงเรียน 
เพื่อให้ได้มุมมองทั้งจากผู้เรียนและผู้สอน

โดยเรามีข้อสันนิษฐานในการสัมภาษณ์นักเรียนเบื้องต้นว่า
\begin{itemize}
    \item นักเรียนสามารถวาดรูปแสดงเศษส่วนเป็นรูปแบบ area model ได้
    \item นักเรียนสามารถวาดรูปแสดงเศษส่วนเป็นรูปแบบอื่นนอกจาก area model ได้
    \item นักเรียนสามารถอธิบายการเปรียบเทียบเศษส่วนได้
    \item นักเรียนสามารถอธิบายถึงการบวกเศษส่วนได้
    \item นักเรียนสามารถระบุตำแหน่งของ $\frac{1}{4}$ บนเส้นจำนวนได้อย่างถูกต้อง
\end{itemize}

จากการสัมภาษณ์นักเรียน พบว่า 
\begin{itemize}
    \item นักเรียนส่วนใหญ่สามารถวาดรูปแสดงเศษส่วนในรูปแบบ area model ได้ แต่มีนักเรียนเพียงส่วนน้อยที่สามารถวาดรูปแสดงเศษส่วนในรูปแบบอื่น เช่น length model หรือ set model ได้
    \item นักเรียนส่วนใหญ่สามารถเปรียบเทียบ และบวกเศษส่วนได้ แต่เมื่อสอบถามถึงเหตุผลเบื้องหลังการคิด โดยเฉพาะในการบวกเศษส่วนว่าเหตุใดจึงต้องแปลงตัวส่วนให้เท่ากันก่อนทำการบวก 
    นักเรียนส่วนใหญ่กลับตอบว่า `เพราะครูสอนมาแบบนี้` ซึ่งสะท้อนให้เห็นว่านักเรียนจำวิธีการมาใช้โดยไม่สามารถอธิบายเหตุผลเบื้องหลังการคิดได้อย่างชัดเจน
    \item นักเรียนส่วนใหญ่ไม่สามารถระบุตำแหน่งของ $\frac{1}{4}$ บนเส้นจำนวนได้อย่างถูกต้อง
\end{itemize}

จากการสัมภาษณ์ครูผู้สอน พบว่า ครูส่วนใหญ่เริ่มต้นสอนเศษส่วนด้วยการยกตัวอย่างจากสิ่งใกล้ตัว เช่น การแบ่งพิซซ่าหรือเค้ก เพื่อให้เด็กมองเห็นภาพได้ง่ายขึ้น 
เนื่องจากก่อนหน้านั้นนักเรียนจะคุ้นเคยกับ “จำนวนเต็ม” มากกว่า การมองเห็นปริมาณที่ไม่เต็มหน่วยจึงเป็นสิ่งที่ท้าทายสำหรับเด็กในช่วงแรก 
และหนังสือเรียนของ สสวท. ก็ยังคงใช้การอธิบายด้วย area model ซึ่งเป็นการแบ่งพื้นที่เป็นส่วน ๆ 
ทำให้เด็กบางคนไม่สามารถเชื่อมโยงแนวคิดนี้ไปสู่การใช้เศษส่วนในบริบทอื่น ๆ ได้ นอกจากนี้ ครูผู้สอนยังให้ความเห็นเพิ่มเติมว่า 
เนื้อหาเรื่องเศษส่วนนั้นมีลักษณะที่เป็นนามธรรมมากเกินไป ทำให้เด็กไทยหลายคนจับต้องและทำความเข้าใจได้ยาก

สิ่งที่คุณครูหลายคนกล่าวตรงกันคือ “ความยากที่สุดของการสอนเศษส่วนคือการทำให้เด็กเห็นภาพว่าเศษส่วนคืออะไร” โดยเฉพาะในเรื่องของการคูณและหาร 
ซึ่งคุณครูมักรู้สึกว่ายากต่อการอธิบายให้เด็กเข้าใจ ขณะที่สำหรับนักเรียนเอง กลับพบว่าการบวกและลบเศษส่วนเป็นหัวข้อที่เข้าใจยากที่สุด และในบางโรงเรียน 
ครูผู้สอนมีการปรับลำดับเนื้อหาการสอนใหม่ เพื่อให้สอดคล้องกับลำดับความเข้าใจของผู้เรียนจริง เนื่องจากหลักสูตรของ สสวท. มีลักษณะการจัดเรียงเนื้อหาที่ไม่ต่อเนื่อง 
ทำให้เด็กบางคนยังไม่พร้อมจะเรียนรู้หัวข้อที่ซับซ้อนขึ้น

ข้อมูลที่ได้จากครูและนักเรียนสะท้อนภาพเดียวกันว่า การสอนเศษส่วนยังขาดสื่อหรือเครื่องมือที่ช่วยให้เด็กเข้าใจเศษส่วนอย่างเป็นรูปธรรม 
ปัญหานี้จึงกลายเป็นจุดเริ่มต้นให้พวกเราคิดต่อยอดไปสู่การพัฒนาสื่อการสอนรูปแบบใหม่

นอกจากนี้ จากการสัมภาษณ์นักเรียนและครูผู้สอน พบว่านักเรียนส่วนใหญ่มีสมาร์ตโฟนเป็นของตนเอง และมักใช้เวลาเล่นเกมหรือสื่อสังคมออนไลน์ เช่น 
TikTok, Roblox, PUBG และ Free Fire เป็นประจำ ซึ่งสะท้อนให้เห็นว่าเทคโนโลยีเป็นส่วนหนึ่งในชีวิตประจำวันของนักเรียน 
พวกเราจึงเห็นว่า นี่เป็นโอกาสที่ดีในการเปลี่ยนเวลาที่เด็ก ๆ ใช้มือถือให้กลายเป็นช่วงเวลาการเรียนรู้นอกห้องเรียน ผ่านการทำสื่อการสอนที่อยู่ในรูปแบบของเทคโนโลยีดิจิทัล

ดังนั้น พวกเราจึงตั้งเป้าหมายที่จะพัฒนา สื่อการสอนในรูปแบบ web application ที่สามารถใช้งานได้ทั้งบนสมาร์ตโฟนและแท็บเล็ต เพื่อให้นักเรียนสามารถเรียนรู้ได้ทุกที่ทุกเวลา 
สื่อการสอนนี้จะเน้นการอธิบายแนวคิดเศษส่วนผ่าน length model โดยอ้างอิงจากงานวิจัย The Effects of Using Length Models to Teach Fraction and Decimal Translation~\cite{rojo2024}
ซึ่งพบว่าการเรียนรู้ด้วย length model ช่วยให้ผู้เรียนเข้าใจความหมายของเศษส่วนได้ชัดเจนกว่าแบบ area model 
และสามารถเชื่อมโยงแนวคิดการใช้เศษส่วนไปในบริบทอื่น ๆ ได้ง่ายกว่าแบบ area model

\section{\ifenglish Objectives\else วัตถุประสงค์ของโครงงาน\fi}
จากปัญหาที่ได้กล่าวมาในหัวข้อ 1.1 เราจึงมีความตั้งใจที่จะจัดทําโครงงานนี้ขึ้นมาเพื่อแก้ไขปัญหาดังกล่าว โดยมีวัตถุประสงค์และขอบเขตของโครงงานดังนี้
\begin{enumerate}
    \item ออกแบบสื่อการสอนโดยใช้แนวคิดของ length model เพื่ออธิบายแนวคิดเศษส่วน
    \item พัฒนาสื่อการสอนในรูปแบบ web application ที่สามารถใช้งานได้ทั้งบนสมาร์ตโฟนและแท็บเล็ต โดยใช้ length model ในการอธิบายแนวคิดเศษส่วน
    \item ทำให้นักเรียนสามารถวาดรูปแสดงเศษส่วนในรูปแบบ length model ได้
    \item ทำให้นักเรียนสามารถอธิบายการเปรียบเทียบเศษส่วนได้อย่างชัดเจน
    \item ทำให้นักเรียนสามารถอธิบายถึงการบวกเศษส่วนได้อย่างชัดเจน
    \item ทำให้นักเรียนสามารถเข้าใจเศษส่วนในระบบเส้นจำนวนได้
\end{enumerate}

\subsection*{ผลกระทบด้านสังคม}
พวกเรามองว่า โครงงานสื่อการสอนเศษส่วนในรูปแบบ web application จะช่วยให้เด็กทุกคนเข้าถึงการเรียนรู้ได้อย่างเท่าเทียม 
ไม่จำกัดเพียงโรงเรียนหรือพื้นที่ใดพื้นที่หนึ่ง ทำให้ลดความเหลื่อมล้ำทางการศึกษาในระดับประถมศึกษาได้ ถือเป็นผลกระทบด้านสังคมรูปแบบหนึ่ง

\subsection*{ผลกระทบด้านสุขภาพ}
พวกเรามองว่า สื่อการสอนนี้มีส่วนช่วยลดความเครียดและความวิตกกังวลในการเรียนวิชาคณิตศาสตร์ของนักเรียนได้ เนื่องจากการมีช่องทางและรูปแบบการเรียนรู้ที่หลากหลายเพิ่มมากขึ้น 
จะช่วยลดความกดดันจากการเรียนแบบเดิม ๆ ในห้องเรียน นอกจากนี้ จากการสังเกตพฤติกรรมของผู้เรียนระหว่างการทดสอบใช้งานพบว่า 
นักเรียนให้ความสนใจและรู้สึกสนุกสนานกับการเรียนรู้ผ่านสมาร์ตโฟนหรือแท็บเล็ต มากกว่าการทำแบบฝึกหัดหรือโจทย์ปัญหาบนกระดาษอย่างเห็นได้ชัด 
ซึ่งการนำเสนอเนื้อหาเรื่องเศษส่วนผ่านสื่อเทคโนโลยีที่เข้าถึงง่ายและเห็นภาพได้ชัดเจนนี้ ไม่เพียงแต่ช่วยให้เกิดความเข้าใจ แต่ยังส่งผลดีต่อสุขภาพจิตและช่วยสร้างทัศนคติเชิงบวกต่อการเรียนคณิตศาสตร์อีกด้วย

อย่างไรก็ตาม การใช้งานสื่อการสอนผ่าน web application บนสมาร์ตโฟนหรือแท็บเล็ต อาจมีผลกระทบด้านสุขภาพร่างกายของผู้เรียนได้ หากใช้งานและจ้องหน้าจอติดต่อกันเป็นเวลานานเกินไป
อ้างอิงจากงานวิจัย Digital Eye Strain-- A Comprehensive Review (@Anak, insert citation here) ที่ได้รวบรวมการศึกษาเกี่ยวกับภาวะตาล้าจากอุปกรณ์ดิจิทัล (digital eye strain หรือ computer vision syndrome) 
โดยระบุว่า การเพ่งมองหน้าจออิเล็กทรอนิกส์อย่างต่อเนื่องเป็นปัจจัยสำคัญที่ก่อให้เกิดภาวะตาล้า อาการปวดตา ตาแห้ง และอาจส่งผลกระทบต่อสุขภาวะทางสายตาในระยะยาวได้ 

นอกจากนี้ การใช้งานอุปกรณ์ในอิริยาบถที่ไม่เหมาะสมติดต่อกันเป็นเวลานาน ยังอาจก่อให้เกิดความผิดปกติของระบบกระดูกและกล้ามเนื้อได้
อ้างอิงจากงานวิจัย Predictors of Musculoskeletal Pain among Primary School Students Using Smartphones in Nakhon Si Thammarat, Thailand (@Anak, insert citation here)
พบว่า การใช้งานสมาร์ตโฟนของนักเรียนระดับประถมศึกษาที่พบว่า การใช้งานในท่าทางที่ไม่ถูกต้อง เป็นปัจจัยเสี่ยงสำคัญที่ทำให้เกิดอาการปวดเมื่อยบริเวณคอ บ่า และไหล่
ดังนั้น การนำสื่อการสอนไปใช้จริงจึงควรมีการให้คำแนะนำแก่ผู้เรียนเกี่ยวกับระยะเวลาและอิริยาบถที่เหมาะสมในการใช้งานควบคู่ไปด้วย เพื่อป้องกันผลกระทบด้านสุขภาพที่อาจเกิดขึ้น

\subsection*{ผลกระทบด้านวัฒนธรรม}
พวกเรามองว่า โครงงานนี้ช่วยส่งเสริมวัฒนธรรมการจัดการเรียนรู้ที่เน้นความเข้าใจผ่านการมองเห็นภาพ (visual learning) มากกว่าการท่องจำแบบเดิมอย่างเป็นรูปธรรม 
จากเดิมที่รูปแบบการเรียนการสอนคณิตศาสตร์ในประเทศไทยมักเน้นการจำกฎเกณฑ์เป็นหลัก เช่น ในเรื่องการบวกเศษส่วน ผู้เรียนมักถูกสอนให้จดจำว่า หากตัวส่วนเท่ากันสามารถนำตัวเศษมาบวกกันได้ทันที 
แต่หากตัวส่วนไม่เท่ากัน จะต้องใช้วิธีแปลงตัวส่วนให้เท่ากันก่อนทำการบวก ซึ่งวิธีการดังกล่าวทำให้ผู้เรียนขาดความเข้าใจในหลักการและไม่สามารถอธิบายถึงเหตุผลเบื้องหลังของกระบวนการนั้นได้
นอกจากนี้ เมื่อพิจารณาเนื้อหาในแบบเรียนคณิตศาสตร์ของสถาบันส่งเสริมการสอนวิทยาศาสตร์และเทคโนโลยี (สสวท.) จะพบว่ายังคงมีการนำเสนอการบวกเศษส่วนในรูปแบบนามธรรม (abstract) ตั้งแต่ต้น 
โดยมุ่งเน้นไปที่การคำนวณผ่านตัวเลขและสัญลักษณ์ แทนที่จะเริ่มต้นจากการใช้สื่อเชิงประจักษ์เพื่อปูพื้นฐานให้ผู้เรียนเห็นภาพว่าสัดส่วนเหล่านั้นสามารถนำมารวมกันได้อย่างไร 
แนวทางดังกล่าวยิ่งเป็นการตอกย้ำให้เห็นว่าระบบการศึกษาไทยยังคงยึดติดกับการสอนแบบท่องจำมากกว่าการสร้างความเข้าใจที่แท้จริง
โครงงานนี้จึงเข้ามาช่วยแก้ไขปัญหาโดยการใช้สื่อเทคโนโลยีเข้ามาอธิบายให้ผู้เรียนเห็นภาพแนวคิดและหลักการของเศษส่วน ทำให้ผู้เรียนเข้าใจถึงเหตุผลเชิงตรรกะว่ากระบวนการบวกเศษส่วนเกิดขึ้นได้อย่างไร 
และเหตุใดจึงต้องปรับตัวส่วนให้เท่ากันก่อน การเปลี่ยนแปลงนี้นอกจากจะช่วยเปลี่ยนผ่านจากการเรียนแบบท่องจำมาสู่การเรียนรู้ด้วยความเข้าใจแล้ว ยังเป็นการกระตุ้นให้เกิดการนำเทคโนโลยีมาบูรณาการในห้องเรียน 
ซึ่งจะช่วยยกระดับรูปแบบการสอนคณิตศาสตร์ระดับประถมศึกษาให้ทันสมัยและมีประสิทธิภาพมากขึ้น
